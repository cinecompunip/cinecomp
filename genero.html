Termo Gênero Cinematográfico


Uma breve explicação, é a forma utilizada no setor para diferenciar os variados estilos de filmes existentes, é bom para o público pois permite determinar qual ou quais os gêneros mais preferem assistir.
Listaremos alguns gêneros e uma breve explicação, porém os participantes deste projeto escolherão alguns gêneros com suas recomendações.

  • Ação
  • Aventura
  • Comédia
  • Documentário
  • Drama
  • Espionagem
  • Faroeste
  • Fantasia
  • Guerra
  • Ficção Científica
  • Musical
